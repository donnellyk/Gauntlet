StateTransitionDelegates

public struct StateTransitionDelegates<T: StateType>

Delegate tasks of StateMachine. Consumers can assign implementations to respond to lifecycle events.

  • didTransition

    Delegate closure that gets called whenever StateMachine successfully transitions from one state to another.

    Warning

    Like all delegates, there’s a high likelihood the delegatee owns the delegator and retain cycles are a danger. If the closure references the delegatee (even as self), it should be declared weak in a capture list. See String Reference Cycles for Closures.
